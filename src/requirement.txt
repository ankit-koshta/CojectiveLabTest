Create a task management web application where users can add, edit, delete, and mark tasks as completed. The application should have a clean and user-friendly interface built with React and Material-UI components.
Requirements:
 
FrontEnd:
1. Add Task: Provide a form to add a new task with fields for title, description, deadline, and priority.
2. Task List: Display a list of tasks with options to view details, edit, delete, and mark tasks as completed.
3. Edit Task: Allow users to edit task details including title, description, deadline, and priority.
4. Mark Task as Completed: Implement a feature to mark tasks as completed, visually distinguishing them from active tasks.
BackEnd:
1. Create well defined localhost database (Mongo or PostGRE or Whichever you have.)
2. Create APIs for adding and listing tasks
 

Technologies:
React (functional components with hooks)
Node 
Material-UI (MUI) for UI components
Optional: React Router for routing, Context API for state management
 
Timeframe: 1 hour
Deliverables:
GitHub repository with the complete source code
